# GridIron

## About

An OGRE-powered map editor for the rapid creation of large 3D grid-based environments.

## Dependencies

* Qt v4.7.4+
* OGRE v1.8.0+

## Building

### Unix

1. qmake
2. make

### Windows (with Visual Studio)

1. qmake -tp vc
2. Use resultant Visual Studio project file to build.
